======================
stop-system.sh
======================

SYNOPSIS
========

.. program:: stop-system.sh

.. code-block:: bash

    stop-system.sh

DESCRIPTION
===========

Stop all running Docker containers by executing ``docker compose down`` in the common configuration directory.
